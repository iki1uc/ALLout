# ALLout – O‑Ebene · iki1uc

## Zweck
ALLout ist die Ausleitungs‑Ebene des 6E‑Systems.
Es transportiert alle Ausgänge, Signale, Räume und META‑Impulse nach außen.
ALLout ist die finale Transfer‑Achse des Systems.

## Struktur
- NAME: ALLout
- ROOT: iki1uc
- SYSTEM: 6E
- MODUL: ALLout
- DIM: O (Output)

## Rolle
ALLout ist die Ausgangs‑Achse.
Es verbindet API (D5), FREQ (S) und GEO (R) zu einem externen Transferpunkt.

## Regeln

- ALLout liegt in der O‑Ebene (Output).
- ALLout darf die 6E‑Matrix nicht verändern.
